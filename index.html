مختبر الناموس البيطري

مختبر الناموس البيطري هو احد المختبرات المتميزة التي تقدم خدمة خاصة لسباقات الهجن والخيول العربية الأصيلة يتميز المختبر بدقه الفحوصات وذالك نسبه لاستخدام احدث الاجهزه البيطرية الموافقه للمواصفات .الدوليه تحت اشراف أطباء متخصصين وفنيين لهم خبره طويله فى مجال التحاليل المحبريه

خدمات المختبر

  • نعمل فحوصات الدم والمناعة والصفايح الدموية بجهاز خاص للحيوانات والفحوصات الكيميائية
  • فحص اللحقاح للخيول والجمال (فحص الحمل )
  • فحص امراض طفيليات الدم مثل البروسيلا
  • فحص زمبور – بابيسيا – ثاليريا – انابلازما
  • فحص البعر خاص – تحاليل الروث ( ديدان الاسطوانية و الشريطية و الكوكسيديات وغيرها )
  • فحص فايتمنات مثل ( B1)
  • اعطاء نتائج الفحوصات دقيقة وبسرعة عالية
  • نقوم بالوصول اليكم لاستلام العينات الدم
  • ارسال النتائج لاصحاب الهجن المشاركة في سباقات بطرق مختلفة ( يدوي والالكتروني والتلفون )
